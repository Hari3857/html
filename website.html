coursename = WEB TECHNOLOGY

duration:21 weeks, fees: 15,000 INR where you can learn the basics of web development including HTML, CSS, and JavaScript. Basics: HTML (structure & semantic tags), CSS (styling, layout, responsive) and JavaScript (DOM, events, interactivity).
HTML: defines page structure — headings, paragraphs, links, forms, semantic elements like header/footer/article.
CSS: controls presentation — selectors, box model, Flexbox/Grid, media queries for responsive design.
JavaScript: adds behavior — manipulate DOM, handle events, make network requests (fetch/API), async programming.
Client & Server: browser sends HTTP requests; server returns responses and APIs (REST/JSON).
Tools & Best practices: use browser DevTools, code editors, Git; follow accessibility, performance, and security (HTTPS) guidelines.
